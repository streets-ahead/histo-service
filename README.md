histo-service
=============

A simple Node.js service to generate histograms, and mostly so I can play with html5 canvas.

A URL like this

	http://localhost:8888/img.png?numbers=%5B4,3,2,6,1,3,5,2,4,3,2,4,3,2%5D&color=c3ce69
	
Will get you
